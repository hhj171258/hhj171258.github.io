Database, Table BackUp

	cd \xampp\mysql\bin
mysqldump -u root -p[password] [database] ([table]) > [file]

Database, Table Restore

	cd \xampp\mysql\bin
mysql -u root -p[password] [database] < [file.txt]

txt -> table load

	cd C:/[folder]/[file.txt]
load data infile 'C:/[folder]/[file.txt]' into table [table] field terminated by '[구분자]'

table -> txt Download

	cd \xampp\mysql\bin
mysql -u root (-p[password]) ([database]);
select * into outfile 'C:/[folder]/[file.txt]' fields terminated by '#' from supplier;
