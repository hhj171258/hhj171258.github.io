SQL

SQL은 Structured Query Language(구조화된 질의어)의 약어로 RDBMS(Relational DataBase Management System)의 데이터를 관리하기 위해 설계된 특수목적의 프로그래밍 언어이다.
RDBMS에는 Oracle Database, MySQL, MS-SQL, MS-Access, Informix, Paradox, Syntax 등이 있다.

Install xampp

xampp를 설치한다. xampp는 Apache, MariaDB(MySQL), PHP, Pearl의 약어이다.

MySQL 실행하기

다음은 xampp control pannel로 Apache와 MySQL을 실행한 뒤 입력한다.

	cd \xampp\mysql\bin
mysql -u root
show databases;
create database [database]; 
use [database];
show tables;
create table [table](
	[column1] [type] not null,
	[column2] [type],
	[column3] [type],
	primary key([colum1])
);
desc [table];

MySQL 종료하기

	quit
exit

MySQL 비밀번호 설정

	use mysql;
set password = password('[password]');
mysql -u root -p[password]

MySQL 비밀번호 변경

	mysql.exe -skip-grant
mysql		
use mysql;
UPDATE user set password = PASSWORD('[password]') where user = 'root';
FLUSH PRIVILEGES;
quit
