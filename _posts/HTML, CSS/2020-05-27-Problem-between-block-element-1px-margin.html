problem

Situation

  • <meta name="viewport" content="width=720px, target-densitydpi=device-dpi">
  • <div> width를 px로 지정
  • <div> height를 지정해 주지 않아 소숫점 존재
  • <div> background image를 cover로 적용

Problem

몇몇 mobile 기기에서 <div> 사이에 1px margin 발생
문제가 발생하는 기기와 브라우저의 공통된 특징 확인 불가

Solution

invalid Solution

  • <div> width를 100%로 변경
  • <div> height 지정
  • <div> background image 삭제

valid Solution

  • <div> parent background-color 지정
  • <div> margin-top: -1px
