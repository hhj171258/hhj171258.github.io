iframe 삽입

	<iframe width="[widht]" height="[height]" src="https://www.youtube.com/embed/" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

원하는 Youtube 영성에서 공유 > 퍼가기에 있는 html 코드를 html에 삽입한다.

반응형 높이 지정

	div{position: relative; width: 100%; height: 0; padding-bottom: 56.25%;}
div iframe{position: absolute; top: 0; left: 0; width: 100%; height: 100%;}

반응형에서 <iframe width="100%" height="auto">로 두면 height가 너무 작아진다. 이를 해결하기 위해 <iframe>을 한 번더 감싼 뒤에 padding-bottom: 56.25%;을 줘야한다. 이때, 56.25%는 100 * (9 / 16)이다.
