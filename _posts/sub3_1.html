Git 기본 명령어

Git 다운로드

우선 Git을 설치한다.

초기 설정

				$ git config --global user.name "[your name]"
$ git config --global user.email "[your email]"

이때 이메일은 github 회원가입에 사용했던 이메일이다.

로컬 저장소 생성

				$ mkdir ~/project
$ cd ~/project
$ git init

mkdir은 make directory의 준말이다. ~/는 컴퓨터 최상위 단계 파일을 뜻한다.
cd는 change directory의 준말로 해당 directory로 이동한다.
init는 initailize의 준말로 저장소가 생성되고 .git folder가 생성된다.

파일 생성 및 커밋

				$ git status$ touch [file]
$ git add [file]
$ git commit -m "제목(영문 70자 이하) 공백 내용"
$ git commit --amend "최근 커밋 메세지 수정"
$ git log

작업 되돌리기

				$ git status
$ git log
$ git checkout [commitID]
$ git checkout [commitID] [file]
$ git checkout master

깃허브 저장소 연결 및 업로드

				$ git remote -v
$ git remote add origin [url]
$ git push origin master

ctrl + v 는 insert를 누르면 된다.

깃허브 저장소 복제

$git clone [url]
