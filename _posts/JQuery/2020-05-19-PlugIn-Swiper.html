많이 쓰이는 배너 PlugIn 중 하나.Owl Carousel와 slick도 많이 쓰인다.

Parameters

	var mySwiper = new Swiper ('.swiper-container',
	direction: horizontal | vertical, // default: horizontal
	speed: [number], // default: 300
	width: [number],
	height: [number],
	slidesPerView: [number] | auto, // default: 1, auto 사용시 .swiper-container에 width(또는 height)에 값을 주면 안된다.
	loop: [boolean], // default: false
	breakpoints: {
		[number]:{
			[Parameter] : [Value]
		}
	},
	spaceBetween: [number] // default: 0, 공식 사이트에서는 margin 사용 지양을 권고한다.
});

Parameter가 다양해서 추가적인 Method는 웬만하면 필요 없을듯하다. 나머지는 Swiper API에서 볼 수 있다.
