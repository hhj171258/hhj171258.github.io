level1 Multiples Of 3 and 5

problem

10미만의 자연수에서 3과 5의 배수를 구하면 3, 5, 6, 9이다. 이들의 총합은 23이다.
1000미만의 자연수에서 3, 5의 배수의 총합을 구하라

solution

				function solution(){
	for(var answer = 0; answer < 1000; answer++){
		if((answer % 3 == 0 )|| (answer % 5 == 0)){
			answer += answer;
		}
	}	
	var result = answer;

	return result;
}
